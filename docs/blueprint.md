# **App Name**: WanderWise

## Core Features:

- Personalized Itinerary Creation: AI-powered itinerary generation based on user preferences, budget, and trip length, including real-time adjustments and drag-and-drop editing using tool.
- Real-time Translation: Text, voice, and image translation with offline phrasebook and auto-detect language feature.
- Cultural AI Chatbot: AI chatbot providing cultural etiquette, local customs, food recommendations and navigation assistance, available offline for cached destinations, using tool.
- Budget & Expense Tracking: Set trip budgets, track expenses by category, and auto-capture spending from receipts.
- Safety Alerts: Location-based safety alerts and real-time weather warnings with emergency numbers by region.
- Offline Access: Downloadable maps, itineraries, and phrasebooks for offline use.
- Ticket and Accommodation Booking: Integration with APIs for booking flights, trains, buses, and accommodations with price comparison.
- Theme Toggle: Allow users to switch between a light and dark theme for optimal viewing experience.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F), evoking nature and tranquility.
- Background color: Light beige (#F5F5DC), creating a soft, neutral backdrop.
- Accent color: Warm orange (#E9967A), used for calls to action and important notifications.
- Body and headline font: 'PT Sans', a modern humanist sans-serif for readability and approachability.
- Simple, minimalist icons for easy navigation and visual appeal.
- Clean, intuitive layout with a focus on ease of use and accessibility.
- Subtle animations and transitions to enhance user experience.
